# Demo Guide

## Target

Finish in 90 seconds.

## Flow

```text
Problem → record voice → AI draft → review/edit → save → weekly snapshot → share worker-confirmed record
```

## Demo Rules

- Use one natural prepared work note.
- Point out that review happens before storage.
- Show a pending-payment follow-up date only as a local aid.
- Call shared records `worker-confirmed`, never verified proof.
- Keep fallback ready but explain that live AI needs internet.
