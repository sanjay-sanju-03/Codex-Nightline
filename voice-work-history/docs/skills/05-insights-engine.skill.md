# Ledger Insights

## Inputs

Confirmed records in IndexedDB.

## Current Outputs

- Total hours, paid amount, and pending amount.
- Current-week hours, paid amount, and pending amount.
- Pending-payment follow-up dates.
- A local pending-payment suggestion based on actual records.

## Rules

- Derive insights deterministically from stored records.
- Do not invent wages, reminders, risk scores, or financial advice.
- Explain that a follow-up date is local and does not message an employer.
